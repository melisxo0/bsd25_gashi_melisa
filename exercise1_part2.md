# Exercise 1 Part 2 - Git Befehle

## git config
Konfiguriert Git mit Name und E-Mail-Adresse.

## git clone
Kopier ein Repository vom Server auf den lokalen Rechner.

## git status
Zeigt den aktuellen Status des Repositories an.

## git add
Fügt Dateien zur Staging Area hinzu.

## git commit
Speichert doe Änderungen dauerhaft im Repository.

## git push
Lädt die lokalen Commits auf den Server hoch.

## git pull
Holt die aktuelle Version vom Server.
